# Oracle + Cursor MCP

This workspace’s MCP file tree does **not** include an **Oracle Cloud Infrastructure** server. Provisioning VMs, VCNs, and instances is done via:

1. **Oracle Console** (browser), or  
2. **`oci` CLI** on your laptop with **`~/.oci/config`** (see **`README.md`** and **`env.local`** flow).

Adding a third-party OCI MCP (if one exists) would be a **Cursor / project** configuration change outside this repo.
