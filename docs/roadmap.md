# SyncScript Roadmap

## Implemented Features ✅

### Core Productivity
- **tasks**: Task management with subtasks, priorities, and event linking
- **events**: Event management with preparation tasks and calendar sync
- **calendar-sync**: Bidirectional calendar synchronization with conflict resolution
- **pinned-events**: Dashboard pinned events rail (max 5 events)

### AI & Automation
- **ai-assistant**: Contextual AI assistant with task generation
- **voice-input**: Speech-to-text input with privacy controls

### Budgeting & Financial
- **manual-budgeting**: Task and event-level budgeting with line items

### Energy & Gamification
- **energy-engine**: Daily energy system with EP conversion and challenges
- **achievements**: Domain-specific achievement system with rarity levels

### Templates & Scripts
- **templates**: Template gallery with versioning and PII detection
- **user-scripts**: Personal script creation from events

### Collaboration
- **friends**: Friendship system with privacy controls
- **projects**: ShareScript collaboration with role-based access

### Calendar Integrations
- **google-calendar**: Google Calendar OAuth integration with two-way sync

### Export System
- **export-system**: Multi-format export with RBAC redaction and templates

### Search & Discovery
- **search**: Global search across tasks, events, and projects

### Settings & Profile
- **profile**: User profile management with privacy settings
- **settings**: Feature flag and integration management

## Coming Soon Features 🚧
